Given there is no official Kiosk Mode for iPads, we work around this using a pre-determined username and password that all kiosk users would use. The iPad is then put into Guided Access mode to lock the iPad to the Horizon Client app. This configuration is likely best suited for use of a known set of users where individual profiles/accounts aren't necessary.
